    🔄 Installation & Teilen
    • Das erste Gerät wird automatisch Hauptgerät
    • Hauptgerät: auf 🔗 Teilen tippen → Link erzeugen & ans andere Gerät senden
    • Gastgerät: Link öffnen → automatisch beigetreten, kein Code nötig
    • Immer zu Homebildschirm hinzufügen – danach ist es eine WebApp
    🎤 Mikrofon
    • Auf 🎤 tippen, um Spracherkennung zu starten (beim ersten Mal zulassen)
    • Text im Eingabefeld prüfen
    • Mit ✅ Übernehmen zur Liste hinzufügen
    📱 Buttons
    • 🗑️ Löschen – Eingabefeld leeren
    • ✅ Übernehmen – Erinnerung zur Liste hinzufügen
    • 📷 Foto – Foto aufnehmen, Text dazu fügen möglich
    • 📤 Export – Liste als Text in die Zwischenablage kopieren
    🧠 Erfassen & Erledigt
    • Erfassen – Erinnerungen eingeben und zur Liste hinzufügen
    • Erledigt – Einträge antippen um sie abzuhaken
    • Abgehakte Einträge werden beim Wechsel auf Erfassen gelöscht
